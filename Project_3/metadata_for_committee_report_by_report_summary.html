Metadata Description

Metadata description of Committee Report by Report Summary File

General Description

These files contain information for each time period during which a committee or organization that has registered with the FEC and filed a financial report during current election cycle and for election cycles back to 2008.

The information covers the time period from January 1 of the non-election year through the most recent report submitted by these committees.

The records (rows) for each committee contains information, updated nightly, based upon reports either received (for electronic filings) or computerized (for paper filings from small committees) through 7 pm the previous day. These totals also incorporate any changes made by committees if any report covering the period is amended.

As part of each financial report, committees must provide a summary of their financial activity in each filing, and these summaries for each reporting period are included here. Generally, committees file reports on a quarterly or monthly basis, but some must also submit a report 12 days before primary elections. During the primary election season, therefore, the period covered by this file may be different for different committees. Committees also typically file 12 days before a general election, and again 30 days after the election.

The presentation here includes total receipts and total disbursements during the quarter, month, etc. for the committee along with the beginning and ending cash-on-hand value and total debts owed by the committee at the end of each reporting period. There are several different reporting structures, depending on what type of organization is submitting financial information. To see one example of these reporting requirements you can look at the summary and detailed summary pages of the FEC Form 3X. Detailed descriptions of each element in the file can be found in the line-by-line instructions.

These files can be large, so we've divided the material into sets based on the committee type. You can download all House and Senate campaign committees, all Presidential campaign committees, all PACs or all Party committees.

First click the "customize data" button and then choose a subset from the drop down menu. If you don't choose a group you'll receive the data for House and Senate campaigns.

Downloads for these data (XML or CSV links from the top of the list) include additional information taken from the summary pages of reports submitted by the committee (e.g. contributions from individuals, independent expenditures made by the committee, etc.). All of the elements requested on pages 2-4 of the paper versions of these reports are included in the file.
